## 📋 概要

現在の営業セグメント分析は「企業規模別（大/中/小）」「エリア別」の2軸のみで、解像度が粗い状況です。

本提案では、以下の観点から営業セグメントを詳細化し、データドリブンな営業戦略の立案を可能にします：

1. **業界カテゴリ別の分析** - 電気工事業界内の細分化
2. **年収層別の分布分析** - インディードなどの求人データから実給与を分析
3. **成長率・伸び予測** - 各セグメントの将来性評価
4. **営業優先順位の明確化** - どこのセグメントを狙うべきかの判断基準

---

## 🎯 提案する分析フレームワーク

### 1. 営業セグメントの多次元分析軸

以下の4つの軸で営業セグメントを分類・分析します：

#### 軸1: 業界カテゴリ（電気工事業界の細分化）

現在の「電気工事業界」を以下のように細分化：

| カテゴリ | 定義 | 特徴 |
|---------|------|------|
| **一般電気工事業** | 建築物の電気設備工事 | 最も一般的な電気工事、市場規模が大きい |
| **特殊電気工事業** | 鉄道、道路、ガス等の特殊設備 | 専門性が高く、単価が高い傾向 |
| **電気通信工事業** | 情報通信設備の工事 | 成長領域、IT関連の需要増加 |
| **ネオンサイン工事業** | 広告用ネオンサイン等 | ニッチ市場、特殊スキルが必要 |
| **電気工事サポート** | 設計・保守・点検 | 技術的専門性が高く、高年収傾向 |

#### 軸2: 企業規模 × 資本金

現在の「従業員数」だけでなく、資本金・売上規模も考慮：

| セグメント | 従業員数 | 資本金 | 特徴 |
|-----------|---------|--------|------|
| **超大企業** | 500名以上 | 5億円以上 | 全国展開、高単価、長期成約 |
| **大手企業** | 100-499名 | 1億-5億円 | 地域展開、標準単価、中長期成約 |
| **中堅企業** | 50-99名 | 5,000万-1億円 | 地域密着、標準単価、中期成約 |
| **中小企業** | 20-49名 | 1,000万-5,000万円 | 地域特化、低単価、短期成約 |
| **小規模企業** | 19名以下 | 1,000万円未満 | 個人事業主クラス、変動大 |

#### 軸3: エリア（都道府県 × 都市規模）

現在の「都道府県別」に加え、都市規模も考慮：

| エリアレベル | 定義 | 特徴 |
|------------|------|------|
| **首都圏大都市** | 東京23区、横浜市、川崎市等 | 求人数最多、競争激化、高年収 |
| **地方中核都市** | 県庁所在地等 | 求人数中程度、競争中程度、標準年収 |
| **地方都市** | その他の都市部 | 求人数少、競争少、年収幅が広い |
| **地方・郊外** | 都市部以外 | 求人数少、専門性重視、年収幅が広い |

#### 軸4: 年収層（求人数ベースの分布）

インディードなどの求人データから、実際の年収分布を分析：

| 年収層 | 範囲 | 求人数分布（予想） | 特徴 |
|--------|------|------------------|------|
| **超高年収** | 800万円以上 | 5-10% | 管理職・特殊技能、難易度高 |
| **高年収** | 600-799万円 | 15-20% | ベテラン・スペシャリスト、標準単価 |
| **標準年収** | 400-599万円 | 40-50% | 一般電気工事士、標準単価 |
| **若手年収** | 300-399万円 | 20-30% | 未経験・若手、低単価 |
| **低年収** | 300万円未満 | 5-10% | 未経験・日給単価、変動大 |

## 📊 実施する分析内容

### 1. カテゴリ別平均給与分析

**実施内容**:
- インディードの電気工事士求人データを収集
- カテゴリ別（一般/特殊/電気通信等）に分類
- 各カテゴリの平均年収・中央値・最大値・最小値を算出
- 時系列で過去1-2年の推移を分析

**出力例**:
```
カテゴリ別平均年収（2024年）
- 一般電気工事業: 平均450万円、中央値420万円
- 特殊電気工事業: 平均580万円、中央値550万円
- 電気通信工事業: 平均520万円、中央値500万円
...
```

### 2. 年収層別求人数分布図

**実施内容**:
- インディードの全電気工事士求人を収集
- 年収層ごとに求人数を集計
- 分布図（ヒストグラム）を作成
- カテゴリ別・エリア別の分布も分析

**出力例**:
```
年収層別求人数分布（2024年）
300万円未満: 120件（8%）
300-399万円: 380件（25%）
400-499万円: 540件（36%）
500-599万円: 300件（20%）
600-699万円: 120件（8%）
700万円以上: 40件（3%）
合計: 1,500件
```

### 3. カテゴリ別成長率分析

**実施内容**:
- 過去2-3年の求人数推移を分析
- 各カテゴリの成長率（CAGR）を算出
- 政府統計（電気工事業界の成長予測）と照合
- 今後5年の伸び予測を作成

**出力例**:
```
カテゴリ別成長率予測（2025-2030年）
- 一般電気工事業: +3-5%/年（脱炭素政策による需要増）
- 特殊電気工事業: +2-3%/年（安定成長）
- 電気通信工事業: +5-8%/年（5G・IoT需要拡大）
...
```

### 4. 営業優先順位マトリックス

**実施内容**:
- 各セグメントの「市場規模（求人数）」「平均単価」「成長率」を評価
- 3軸のマトリックスを作成
- 優先順位（S/A/B/C）を決定
- 営業リソース配分の指針を作成

**評価軸**:
- **市場規模**: 求人数が多いほど高評価
- **単価ポテンシャル**: 平均年収が高いほど単価が高くなる可能性
- **成長率**: 将来性が高いほど高評価

**出力例**:
```
営業優先順位マトリックス
S級（最優先）:
- 一般電気工事業 × 大手企業 × 首都圏 × 標準年収層
  → 市場規模:大、単価:中、成長率:中

A級（積極）:
- 電気通信工事業 × 中堅企業 × 首都圏 × 高年収層
  → 市場規模:中、単価:高、成長率:高

B級（選択的）:
...

C級（限定的）:
...
```

## 🔍 データ収集・分析手法

### 1. インディード求人データの収集

**実施方法**:
1. インディードの電気工事士求人をスクレイピング
2. 以下の情報を収集：
   - 求人名・企業名
   - 年収・給与情報
   - 勤務地（都道府県・市区町村）
   - 企業規模（従業員数、資本金）
   - 業種・カテゴリ
   - 求人数（時系列データ）

**既存スクレイピング機能の活用**:
- `src/job_data/scraper.py` の既存スクレイパーを拡張
- インディード専用のスクレイパーを追加

### 2. データ分析・可視化

**実施方法**:
1. Pythonでデータ分析スクリプトを作成
2. 以下の分析を実施：
   - カテゴリ別平均年収算出
   - 年収層別分布の集計
   - 時系列推移の分析
   - 成長率（CAGR）の算出

**出力形式**:
- Markdown形式のレポート
- CSV形式のデータ
- グラフ・チャート（必要に応じて）

### 3. 政府統計データとの照合

**参照データ**:
- 厚生労働省「電気工事業の現状」（就業者数、企業数）
- 経済産業省「建設業統計調査」（業種別統計）
- 総務省「経済センサス」（企業規模別統計）

**照合内容**:
- 収集データと統計データの整合性確認
- 統計データから成長率予測を補完

## 💡 想定される課題と対策

### 課題1: インディードのスクレイピング制約

**対策**:
- 既存のスクレイピング機能を活用
- 利用規約を遵守
- 必要に応じて手動収集も検討

### 課題2: カテゴリ分類の精度

**対策**:
- 求人名・企業名から自動分類を試みる
- 分類が難しい場合は手動分類も検討
- 分類ルールを明確に定義

### 課題3: 年収データの不完全性

**対策**:
- 年収幅がある場合は中央値を採用
- 日給・月給の場合は年収に換算
- データが不十分な場合は注記
