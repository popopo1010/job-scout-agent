# 電気工事施工管理人材紹介事業 戦略

## 事業概要

電気工事施工管理技士（1級・2級）の転職支援を専門とする人材紹介事業です。電気工事に特化した施工管理資格に焦点を当てています。

### 対象資格
- **1級電気工事施工管理技士**: 大規模工事の施工管理を担当
- **2級電気工事施工管理技士**: 中規模工事の施工管理を担当

### 事業の特徴
- **単価**: 240万円/内定（経営Pより）
- **市場規模**: 電気工事市場の成長に伴い拡大予測
- **展開率**: 20%（CA 1人あたり20名リード → 4名展開）
- **粗利益率**: 80%（想定）
- **営業利益率**: 目標40%以上

### 事業の位置づけ
- **FY26**: PMF達成目標
- **電気工事士事業に続く第二の柱**として位置づけ
- 電気工事士から施工管理へのキャリアアップ支援も含む

## USP（独自の売り）

**「データドリブン×AI活用×電気工事施工管理専門」による、展開率20%を実現する唯一の人材紹介サービス**

### 5つのUSP要素

1. **展開率20%の実現** - セグメント掛け合わせ分析による期待展開率の正確な算出（CA 1人あたり20名リード → 4名展開）
2. **AIによるCAスキル向上の自動化** - 営業通話の自動分析と構造化されたフィードバック（電気工事施工管理特化）
3. **新規法人・新規エリアの自動検出** - 機会損失ゼロを実現
4. **電気工事施工管理専門×データ分析の融合** - 専門性とデータ分析の両立
5. **電気工事士からのキャリアアップ支援** - 電気工事士から施工管理へのキャリアパス支援
