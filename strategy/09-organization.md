# 組織体制・役割分担

## 1. 組織体制（FY25）

| 役割 | 人数 | 主な責任 |
|------|------|---------|
| **事業責任者** | 1名 | 事業全体の統括、戦略的意思決定 |
| **CA（キャリアアドバイザー）** | 3名 | 求職者対応、マッチング、成約獲得 |
| **Sales（営業）** | 1名 | 企業開拓、関係構築 |
| **開発・エンジニア** | TBD | AIエージェントシステムの開発・運用 |
| **マーケティング** | TBD | リード獲得、ブランディング |

## 2. 役割分担

### 経営層

- 戦略的意思決定
- リソース配分
- ステークホルダー管理

### CA（キャリアアドバイザー）

- 求職者対応
- マッチング実行
- 成約獲得

### Sales（営業）

- 企業開拓
- 関係構築
- 案件獲得

### 開発・エンジニア

- システム開発
- データ分析
- システム運用
