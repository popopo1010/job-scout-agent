# 詳細メモ

このディレクトリには、プロジェクトの詳細設計メモを格納します。

## ディレクトリ構成

```
notes/
├── README.md                    # このファイル
├── ai-agent-instruction.md      # AIエージェント使用概要書
├── api-implementation/          # API実装に関するメモ
├── integration-design/          # 統合設計に関するメモ
└── misc/                        # その他のメモ
```

## 各ファイルの説明

### ai-agent-instruction.md
AIエージェント（Claude）に対するインストラクション。プロジェクトの全体像、各システムの要件、実装方針を明確に記載。

### api-implementation/
外部API（Claude、Slack、求人サイト等）の実装に関する詳細メモ。

### integration-design/
システム間の統合設計に関するメモ。

### misc/
その他の設計メモ、検討事項、決定事項など。
