# 月次財務シミュレーション

## 1. 前提条件

### 1.1 基本パラメータ

| 項目 | 値 | 備考 |
|------|-----|------|
| **CA 1人あたり対応リード数** | 20名/月 | 固定 |
| **CA 1人あたり展開数** | 4名/月 | 展開率20%（20名×20%） |
| **CA人件費（固定）** | 30万円/月 | 基本給 |
| **CAインセンティブ率** | 10% | 生み出した利益の10% |
| **RA人件費** | 30万円/月 | 固定 |
| **マーケティングコスト** | 100万円/月 | リード獲得費用 |
| **オフィス代** | 10万円/月 | 固定費 |
| **その他ツール代** | 10万円/月 | 固定費 |
| **システム開発費** | 150万円/月 | 開発・保守費用 |
| **平均単価** | 175万円 | 1内定あたりの手数料 |

### 1.2 計算式

```
月リード数 = CA人数 × 20名
月内定数 = CA人数 × 4名
月売上 = 月内定数 × 平均単価175万円

CA人件費（固定） = CA人数 × 30万円
CAインセンティブ = 月粗利益 × 10%
CA人件費（総額） = CA人件費（固定） + CAインセンティブ

粗利益 = 月売上 × 80%（粗利益率）
営業費用 = マーケティング費 + システム開発費 + RA人件費 + オフィス代 + ツール代 + CA人件費（総額）
営業利益 = 粗利益 - 営業費用
```

## 5. 追加コストの検討

### 5.1 現状の固定費

| 項目 | 金額（万円/月） | 備考 |
|------|---------------|------|
| **システム開発費** | 150 | 開発・保守費用 |
| **RA人件費** | 30 | 固定 |
| **オフィス代** | 10 | 固定費 |
| **ツール代** | 10 | 固定費 |
| **合計** | **200** | |

### 5.2 追加検討が必要なコスト

#### 5.2.1 事業責任者・管理部門人件費
- **推奨**: 100-150万円/月
- **理由**: 事業統括、戦略的意思決定、管理業務
- **影響**: 営業利益率に-5-7%の影響

#### 5.2.2 マーケティング担当者人件費
- **推奨**: 50-80万円/月（CA 5名以上の場合）
- **理由**: CA 5名以上になると、マーケティング業務が増加
- **影響**: 営業利益率に-2-3%の影響

#### 5.2.3 RA追加人件費（CA 5名以上の場合）
- **推奨**: 30万円/月（RA 1名追加）
- **理由**: CA 5名で月内定20名の場合、RA 1名では対応困難
- **影響**: 営業利益率に-1-2%の影響

#### 5.2.4 マーケティング費の増額（CA 5名以上の場合）
- **推奨**: 150-200万円/月
- **理由**: CA 5名で月リード100名の場合、マーケ費100万円では不足
- **影響**: CACは上昇するが、規模の経済により営業利益率は維持

#### 5.2.5 その他コスト
- **保険料**: 5-10万円/月（労災保険、社会保険等）
- **通信費**: 5-10万円/月（電話、インターネット等）
- **消耗品費**: 2-5万円/月（文房具、印刷等）
- **研修費**: 5-10万円/月（CA研修、システム研修等）
- **その他雑費**: 5-10万円/月

**合計追加コスト**: 約30-50万円/月

## 6. 推奨シナリオ

### 6.1 段階的拡大シナリオ

## 8. コメント・推奨事項

### 8.1 追加検討が必要なコスト

1. **事業責任者人件費**: 100-150万円/月（必須）
2. **マーケティング担当者**: CA 5名以上の場合、50-80万円/月
3. **RA追加**: CA 5名以上の場合、30万円/月
4. **マーケティング費増額**: CA 5名以上の場合、150-200万円/月
5. **その他コスト**: 保険、通信、消耗品等、30-50万円/月

### 8.2 最適化のポイント

1. **展開率の向上**: 20% → 25-30%に向上させると、営業利益が大幅に増加
2. **単価の向上**: 175万円 → 200万円に向上させると、営業利益率が向上
3. **マーケティング効率化**: CACを下げることで、営業利益率が向上
4. **固定費の最適化**: システム開発費、オフィス代等の最適化

### 8.3 リスク要因

1. **展開率の低下**: 20%を下回ると、営業利益が大幅に減少
2. **単価の下落**: 175万円を下回ると、営業利益率が低下
3. **マーケティング費の増加**: CACが上昇すると、営業利益率が低下
4. **CAの離職**: CA 1名の離職で、月売上が700万円減少

### 8.4 推奨アクション

1. **Phase 1（CA 1-2名）**: 事業検証、展開率20%の達成確認
2. **Phase 2（CA 3名）**: FY25目標達成、PMF確認
3. **Phase 3（CA 5名）**: 規模の経済の実現、マーケティング費増額
4. **Phase 4（CA 10名）**: 最大規模での運営、組織体制の拡充
