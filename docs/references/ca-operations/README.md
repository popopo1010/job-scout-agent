詳細は [`HOW_TO_ADD_FILES.md`](./HOW_TO_ADD_FILES.md) を参照してください。

## クイックガイド

1. **ファイルを配置**: このディレクトリにPDFファイルをコピーまたは移動
2. **Gitに追加**: `git add docs/references/ca-operations/ファイル名.pdf`
3. **コミット**: `git commit -m "Add PDF: ファイル名.pdf"`
4. **プッシュ**: `git push`

## ファイル命名規則（推奨）

- スペースの代わりにアンダースコア（_）またはハイフン（-）を使用
- 日付を含める場合は `YYYY-MM-DD` 形式を使用
- 例: `KJA-基本的なマッチングの流れとポイント_v1.1.pdf`
